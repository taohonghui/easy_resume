# easy_resume

我们的目标是解决重复填写的工作量，自动填充所有的简历！！

快速实现一个网页填充器

### 实现步骤

1 找到 input 元素
1.1 递归 domtree 找到对应的键值对 ｜｜ 难点：正则书写 特殊情况对处理
2 插入模版文件的值 ｜｜ 时间的转换
3 兼容性测试 ｜｜ 主要是时间值的转换
4 chrome 插件集成
